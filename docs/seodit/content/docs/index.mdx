seodit is a blackbox SEO assertion layer for Playwright. Use granular matchers like `toHaveCanonical` and pure HTML/JSON-LD helpers for co-located `.seodit.spec.ts` files.

## Install

```bash
npm install @sargonpiraev/seodit
```

## Features

- Granular Playwright matchers mapped to Next.js metadata concepts
- Pure `assert*` helpers over rendered HTML
- JSON-LD type checks
- `extendSeoditExpect` — no duplicate `@playwright/test` instance
