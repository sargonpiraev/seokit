title: Overview
description: Playwright SEO matchers for Next.js + next-intl apps
---

**seodit** is a Playwright extension with SEO matchers for Next.js apps that use next-intl.

Colocate `page.seodit.spec.ts` next to a route, loop your next-intl `routing.locales`, and assert canonical, hreflang, Open Graph, Twitter, robots, and JSON-LD with granular `toHave*` matchers.

Start with [Getting started](/docs/getting-started).
